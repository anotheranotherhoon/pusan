## 구현 기능

### 기능 1
Redux Toolkit 상태관리 라이브러리를 통한 다크 모드
* localStorage 에 theme의 값( dark 또는 light)
### 기능 2
scroll To Top

* 일정 길이 이상 스크롤이 내려갈 경우 TOP 버튼이 생기고 클릭시 최상단으로 올라가게됨

### 기능 3
firebase를 통한 회원가입

### 기능 4
회원에 따른 wishToGo리스트
* wishToGo리스트를 개별적으로 관리한다.
